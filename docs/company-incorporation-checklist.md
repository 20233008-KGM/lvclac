# 주식회사 설립 체크리스트

기준일: 2026-07-09

이 문서는 `lvclac`를 유료 서비스로 운영하기 위해 주식회사 설립, 사업자등록, 결제 정산, 운영 문구를 준비하는 실행 체크리스트다. 법률/세무 자문은 아니며, 최종 제출 전에는 법무사 또는 세무사에게 확인한다.

## 쉬운 요약

주식회사를 만든다는 것은 "서비스를 파는 이름표와 통장, 세금 신고 주소를 나라에 등록하는 일"이다. Paddle 결제 코드는 sandbox에서 먼저 테스트할 수 있지만, 실제 유료 판매는 회사 정보와 환불/문의/정산 정보가 준비된 뒤 여는 편이 안전하다.

## 먼저 결정할 것

| 결정 | 예시 | 담당 | 예상 소요 |
| --- | --- | --- | --- |
| 회사명 | 주식회사 Farfield Software 또는 새 한글 상호 | manual | 0:30 |
| 본점 주소 | 자택, 공유오피스, 임대 사무실 중 하나 | manual | 0:30 |
| 대표자/임원 | 1인 대표 또는 이사/감사 구성 | manual | 0:20 |
| 자본금 | 초기 운영비와 등기 비용을 감당할 금액 | manual | 0:20 |
| 사업 목적 | 소프트웨어 개발, 정보서비스, 전자상거래 등 | manual + advisor | 0:40 |
| 정산 계좌 방향 | 법인 계좌, Paddle payout 정보 | manual | 0:30 |

## 사업자등록 준비서류

국세청 기준으로 영리법인 본점 사업자등록에는 보통 다음이 필요하다.

- 법인설립신고 및 사업자등록신청서
- 법인 명의 임대차계약서 사본, 임차한 경우
- 주주 또는 출자자 명세서
- 사업허가/등록/신고필증, 해당 업종인 경우
- 현물출자명세서, 현물출자법인인 경우
- 자금출처 명세서, 해당 업종인 경우

사업자등록은 사업장마다 필요하며, 사업개시 전 또는 사업을 시작한 날부터 20일 이내 신청해야 한다.

| 사업자등록번호 | 약관, 개인정보처리방침, 통신판매 표시 | auto after manual input | 0:20 |
| 통신판매업 신고번호 | 푸터 또는 운영 페이지 | auto after manual input | 0:20 |

| Paddle live profile | Paddle dashboard | manual | 0:40 |

## 최소 사용자 할 일

사용자가 직접 해야 하는 일만 따로 모으면 다음이다.

| 할 일 | 이유 | 예상 소요 |
| --- | --- | --- |
| 회사명, 주소, 대표자/주주/자본금 결정 | 법적 책임 정보라 에이전트가 대신 결정하면 안 됨 | 1:40 |
| 인증서, 인감, 통장, 등기 비용 준비 | 외부 인증/금융/납부가 필요함 | 1:40 |
| 법무사/세무사 최종 확인 | 사업 목적, 세무, 4대보험은 상황별 판단이 필요함 | 1:00 |
| 사업자등록과 통신판매업 신고 제출 | 계정 주인과 대표자 인증이 필요함 | 1:40 |
| Paddle live 정산 정보 입력 | 법인 계좌와 세금 정보가 필요함 | 0:40 |

## 주의할 위험

| 위험 | 쉬운 설명 | 대응 |
| --- | --- | --- |
| 금융투자자문 오해 | 계산기가 투자 조언처럼 보이면 문제가 될 수 있음 | 약관/안내문에 계산 보조 도구임을 명확히 표시 |
| 통신판매업 신고 누락 | 온라인 유료 판매 표시 의무가 빠질 수 있음 | 정부24/관할 구청/전문가 확인 후 신고 여부 결정 |
| Paddle와 법인 정보 불일치 | 결제 영수증/정산 정보와 앱 운영자 정보가 다르면 고객이 혼란스러움 | live 전환 전 회사명, 주소, 문의처를 맞춘다 |
| 세무/4대보험 누락 | 대표 급여, 직원 여부에 따라 신고가 달라짐 | 세무사에게 대표 급여/무보수 여부를 먼저 확인 |
