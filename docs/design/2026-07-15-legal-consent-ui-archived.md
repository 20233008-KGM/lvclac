## 보관 목적

회원가입 약관, 개인정보 선택, 공개 법무 문서, 계정 삭제 요청 UI를 한 번에 정리한 검토안이다. 실제 앱 컴포넌트로 구현해 데스크톱과 모바일에서 확인했으나 최종 디자인으로 채택하지 않았다.

이 문서는 아이디어와 검토 결과만 보관한다. 현재 앱 코드, 라우트, 분석 초기화, 회원가입 및 마이페이지 동작에는 이 시안이 적용되어 있지 않다.

## 시안 구성

### 회원가입

- 이용약관 동의와 개인정보 처리방침 확인을 별도 필수 항목으로 표시
- Google 가입과 이메일 가입이 같은 확인 상태를 공유
- Google 공식 색상 로고와 `Google로 계속하기` 문구 유지
- 가입 화면 하단의 중복 약관 링크 제거

### 개인정보 선택 배너

- `필수만 사용`과 `모두 허용`을 동일한 시각적 무게로 배치
- 필수 저장소와 선택 분석 도구를 구분해 설명
- 분석 동의 전 Vercel Analytics와 GA4 초기화를 보류하는 구조
- 세부 설정을 펼쳐 목적을 확인할 수 있는 방식

### 공개 법무 문서

- 이용약관, 개인정보 처리방침, 환불 정책은 별도 URL 유지
- 문서 전환 메뉴, 현재 공개본 상태, 버전, 시행일 표시
- 데스크톱 고정 목차와 모바일 문서 전환 메뉴 제공
- 조항 번호와 본문을 분리한 문서형 레이아웃

### 계정 삭제 요청

- 마이페이지의 작은 삭제 링크에서 확인 대화상자를 여는 흐름
- 즉시 삭제가 아니라 지원 이메일을 통한 삭제 요청임을 명시
- 계정, 클라우드 저장값, 법정 보관 기록의 처리 범위를 설명
- 위험 동작에서는 취소 버튼에 초기 초점을 두는 방식

### 푸터

- 확인된 운영 주체와 지원 이메일만 표시
- 대표자, 주소, 사업자등록번호처럼 확정되지 않은 값은 표시하지 않음

## 검토 결과

- 실제 앱 적용 후보로는 채택하지 않음
- 구현했던 앱 소스 변경과 개발 전용 검토 라우트는 모두 제거함
- 향후 재검토 시 이 문서를 참고하되, 기존 시안을 그대로 복구하지 않고 새로운 디자인 검토부터 시작함
