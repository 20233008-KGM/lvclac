# Free vs Pro 기능

## Free (현재 유지)

- 평가·주문 모드 전체 계산 기능
- 비로그인 사용 가능
- 6개 AdSense 슬롯 (`src/components/PageShell.tsx`)
- 로그인 시 입력값 1세트 저장 (localStorage / Supabase)

## Pro (유료)

| 기능 | 가치 | 구현 난이도 |
|------|------|-------------|
| **광고 제거** | 6슬롯 제거, 집중 UI | 낮음 — `user.isPro` 가드 |
| **클라우드 동기화** | 기기 간 입력값·프리셋 동기화 | 중간 — Supabase 어댑터 완성 필수 |
| **저장 프리셋** | "KOSPI200 롱", "BTC 숏" 등 이름 붙여 저장 (무제한) | 중간 |
| **시나리오 비교** | 2~3개 포지션 나란히 비교 | 중간 |
| **민감도 분석** | 가격 ±N% 슬라이더로 청산가·위험도 즉시 갱신 | 낮음 — 기존 `calc/` 재사용 |
| **결과보내기** | PNG/PDF 스냅샷 (매매 일지용) | 중간 |
| **청산 근접 알림** | 설정 % 이내 접근 시 이메일/푸시 (가격 피드 필요) | 높음 — Phase 3 |

## 핵심 원칙

기본 청산가 계산은 무료 유지. Pro는 **반복 사용 편의 + 리스크 관리 + 깔끔한 UI**.

## Phase별 기능 배치

### Phase 1 — 기반

- 광고 제거
- Stripe 결제·구독 관리

### Phase 2 — Pro 가치 기능

- 저장 프리셋 CRUD
- 시나리오 비교 뷰 (2~3 컬럼)
- 민감도 슬라이더 (`evaluate` 모드 확장)
- 결과 PNG보내기

### Phase 3 — 고급 (선택)

- 청산 근접 알림 (가격 API + 이메일)
- 다중 포지션 포트폴리오 뷰
- 브로커별 템플릿 프리셋 (키움, 미래에셋 등)
